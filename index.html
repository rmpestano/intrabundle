Using Intrabundle

  1. Download jboss forge distribution zip HERE.
  2. Unzip and start Forge by executing FORGE_DIR/bin/forge - more details see HERE
  3. After starting forge, install intrabundle by executing the command: forge git-plugin https://github.com/rmpestano/intrabundle.git
  4. You are ready to introspect OSGi projects, watch the video below
